How to Choose a Theme

  1. Click PaperMod Theme or Academic Theme below to preview each theme individually.
  2. Click Side-by-Side to compare both themes at the same time.
  3. Review how each theme presents the site's content—header, navigation, team profiles, and publications.
  4. When you have decided, follow the Next Steps section below to apply the theme.

Now What? – Next Steps After Choosing a Theme

  1. Set the theme in config.toml — open config.toml at the root of the repository and set the theme field to your chosen theme name: 
    # For PaperMod:
theme = "PaperMod"

# For the Academic / Wowchemy theme:
theme = "academic"
  2. Install the theme as a Git submodule — run one of the following commands from the repository root in your terminal: 
    # PaperMod
git submodule add https://github.com/adityatelange/hugo-PaperMod themes/PaperMod

# Academic / Wowchemy
git submodule add https://github.com/wowchemy/starter-hugo-academic themes/academic
  3. Preview the real site locally — with Hugo installed, run hugo server -D in the repository root and open http://localhost:1313 in your browser.
  4. Commit and push your changes — once you are happy with the look, commit config.toml and the new themes/ submodule and push to GitHub: 
    git add config.toml .gitmodules themes/
git commit -m "Apply chosen Hugo theme"
git push
  5. Enable GitHub Pages — in your repository settings, go to Pages → Source and select the branch and folder that Hugo publishes to (typically gh-pages or /docs depending on your workflow). Your site will then be live at https://theblackstonecenter.github.io/.
