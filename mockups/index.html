How to Choose a Theme

  1. Click PaperMod Theme or Academic Theme below to preview each theme individually.
  2. Click Side-by-Side to compare both themes at the same time.
  3. Review how each theme presents the site's content—header, navigation, team profiles, and publications.
  4. When you have decided on a theme, note its name and update config.toml with the corresponding theme setting.
