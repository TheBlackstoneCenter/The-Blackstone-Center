✅ Academic Theme Applied — What's Next

The Academic theme has been set in config.toml and the site homepage (index.html) now uses the Academic design. The remaining steps to get the full Hugo-powered site running are below.

  1. Enable GitHub Pages — in your repository settings, go to Settings → Pages → Build and deployment → Source and select GitHub Actions, then click Save. Your site will be live at https://theblackstonecenter.github.io/The-Blackstone-Center/ within ~60 seconds.
  2. Install the Academic Hugo theme (optional — for the full Hugo build) — run this from the repository root in your terminal: 
    # Academic / Wowchemy
git submodule add https://github.com/wowchemy/starter-hugo-academic themes/academic
  3. Preview the full Hugo site locally — with Hugo installed, run hugo server -D in the repository root and open http://localhost:1313 in your browser.
  4. Commit and push any content changes — after editing content files, push to GitHub: 
    git add .
git commit -m "Update site content"
git push
